- Rodar o Setup do Trade no cliente (Copiar da sua máquina), lembrando que a pasta que for gerada sempre tem que ser no disco local do cliente (normalmente é o disco C)
- Após isso, será necessário instalar o SQL, vá na pasta de instaladores > SQL e rodar o Setup do SQL
- Selecionar a opção (New SQL Server Standalone Installation or add Features an existing Installation)
- Desativar o FIrewall
- Clicar em next e aceitar os termos do sistema
- Desmarcar a opção do Azure e clicar em Next
- Marcar a opção Default Instance > next
- Na pagina de usuário, coloca o usuário se senha do banco (sa…) e next ate o final
- Apos isso só confirmar se o SQL está funcionando corretamente no Service

BANCO

- Publisoft > backup > colocar o arquivo .bak dentro desta pasta
- Publisoft > instaladores > MsManager
- Nega tudo que aparecer > services > restore database
- Server name - (Caso for local não precisa mexer, caso for TS colocar o IP)
- Ativar SQL Server Authentication
- Username e senha
- Database Name - Nome do banco (TRADE normalmente)
- Confirmar até a parte do ADD Item
- Selecionar o arquivo .bak posto na parta de backup
- Confirmar tudo e clicar Run
- Ao finalizar pode fechar a tela e selecionar o banco na aba esquerda
- Ir no final a abrir o Server Objects > logins > sa
- Verificar se o idioma está em ingles
- Abrir Trade atualizador > configuração > colocar o nome do banco
- Ir na configuração inicial do Trade > + >Configuração Trade
