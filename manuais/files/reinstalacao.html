Reinstalação

Faça um backup do banco de dados (Trade > Utilitários > Banco de Dados > Backup).
Comprima a pasta Publisoft (ou caso use o instalador comprima as pastas Foods, Plugins, Relatórios e Certificados para preservar as customizações do cliente).
Copie a pasta comprimida, cole-a ao novo servidor e extraia a mesma.
Instale o SQL Server – visualize o manual de instalação aqui.
Anexe (Attach) o banco de dados
- Visualize o manual para anexar usando o SQL Lite aqui - Visualize o manual para anexar usando o SQL Server Management aqui.
Configure o ODBC e o usuário – visualize o manual de instalação aqui.
Instale o Certificado Digital no Servidor novo
Certifique-se de atualizar o sistema para as versões mais recentes
Certifique-se de copiar e executar os arquivos .reg que estão no diretório da pasta Trade
Se houver, mapeie o acesso de demais computadores no novo servidor
Configure o ACBr – visualize o manual de instalação aqui.
Se necessário, configure a logo no ACBr – visualize o manual de configuração aqui.
Se o cliente tiver contratado o PubliBackup, configurar o mesmo - visualize o manual de configuração aqui.
